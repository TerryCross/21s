

21s/BlackJack Console Game
--------------------------

Install with:

    $ git clone https://github.com/TerryCross/21s

Change dir/folder to new dir called "21s"

Run:

    $ npm install


To play the game on a local console in node: 

          $ npm run play
    
To run on the web do:

          $ npm run webplay
          
To run tests:
       
         $ npm test
